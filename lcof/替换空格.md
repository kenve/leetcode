### [面试题 05. 替换空格](https://leetcode-cn.com/problems/ti-huan-kong-ge-lcof/)

**题目：** 请实现一个函数，把字符串 `s` 中的每个空格替换成"%20"。

示例 1：

```
输入：s = "We are happy."
输出："We%20are%20happy."
```

限制：`0 <= s 的长度 <= 10000`

**题解一：正则**

```js
/**
 * @param {string} s
 * @return {string}
 */
var replaceSpace = function(s) {
  return s.replace(/ /g, '%20');
};
```

**题解二：数组**

```js
/**
 * @param {string} s
 * @return {string}
 */
var replaceSpace = function(s) {
  return s.split(' ').join('%20');
};
```
